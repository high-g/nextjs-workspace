ROADMAP.md と CLAUDE.md を現在の進捗状況に合わせて更新し、README.md も最新の状態に反映する。

## 手順

1. ROADMAP.md を読み込む
2. CLAUDE.md を読み込む
3. README.md を読み込む
4. 現在の会話の内容・完了したタスク・次にやることを把握する
5. ROADMAP.md を更新する
   - 完了したタスクを `- [x]` にチェック
   - 新しいタスクがあれば追加
6. CLAUDE.md を更新する
   - `## 現在の状況` セクションを最新の状態に書き換える
   - 解決済みの問題を追記
   - 次回やることを具体的な手順付きで記載する
7. README.md を更新する
   - `## 進捗` テーブルを ROADMAP.md のチェック状況に合わせる（全チェック済み → 完了、未チェックあり → 未着手または進行中）
   - `## 構成` を CLAUDE.md の構成に合わせる
   - `## スタック` を現在のスタックに合わせる

## 注意事項

- ROADMAP.md / CLAUDE.md / README.md 以外のファイルは編集しない
- コードや設定ファイルは変更しない（CLAUDE.md のルールに従う）
- 次回やることのコードスニペットは CLAUDE.md に残しておく
- セットアップ手順（Docker / DevContainer / ローカル）は変えない
