ROADMAP.md と CLAUDE.md を現在の進捗状況に合わせて更新する。

## 手順

1. ROADMAP.md を読み込む
2. CLAUDE.md を読み込む
3. 現在の会話の内容・完了したタスク・次にやることを把握する
4. ROADMAP.md を更新する
   - 完了したタスクを `- [x]` にチェック
   - 新しいタスクがあれば追加
5. CLAUDE.md を更新する
   - `## 現在の状況` セクションを最新の状態に書き換える
   - 解決済みの問題を追記
   - 次回やることを具体的な手順付きで記載する

## 注意事項

- ROADMAP.md と CLAUDE.md 以外のファイルは編集しない
- コードや設定ファイルは変更しない（CLAUDE.md のルールに従う）
- 次回やることのコードスニペットは CLAUDE.md に残しておく
