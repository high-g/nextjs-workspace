ROADMAP.md と CLAUDE.md の内容を元に README.md を最新の状態に更新する。

## 手順

1. ROADMAP.md を読み込む
2. CLAUDE.md を読み込む
3. README.md を読み込む
4. 現在の進捗・構成・スタックを把握する
5. README.md を更新する
   - `## 進捗` テーブルを ROADMAP.md のチェック状況に合わせる（全チェック済み → 完了、未チェックあり → 未着手または進行中）
   - `## 構成` を CLAUDE.md の構成に合わせる
   - `## スタック` を現在のスタックに合わせる

## 注意事項

- README.md 以外のファイルは編集しない
- コードや設定ファイルは変更しない（CLAUDE.md のルールに従う）
- セットアップ手順（Docker / DevContainer / ローカル）は変えない
